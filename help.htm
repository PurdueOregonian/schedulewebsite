To set the schedule, set each box to how many employees should work at each half-hour interval of the week. For example, if 3 employees should work at 7:00am-7:30am on Sunday, set the Sunday 7:00a box to 3.

Add One

The Add One button adds one employee to all selected boxes:

The Subtract One button will subtract one employee from all selected boxes with at least 1 employee.

To add an employee, click "Add Employee" and put in the desired values.

Employees can be set to different priorities. In case of conflict, lower priority employees will be more likely to be scheduled for less than their minimum or more than their maximum number of hours.

The greater the number, the higher the priority.

"Minimum Hours" is the sum of minimum hours for all employees. If the total employee hours is less than this, the schedule will be impossible to satisfy and creating the schedule will give an error.

"Maximum Hours" is the sum of maximum hours for all employees. If the total employee hours is greater than this, the schedule will be impossible to satisfy and creating the schedule will give an error.

Removing Employees

"Remove Employees" deletes all the employees where the checkbox on the left is checked.

Setting Availability

Clicking "Set Availability" on any employee will open the following window. Green cells indicate that the employee is available to work at that time:

The "Set Unavailable" button makes the employee unavailable for the selected time. This is shown in red. When the schedule is created, the program will do its best to avoid scheduling the employee for that time.

The "Set Must Work" button sets a shift that the employee must be scheduled to work. In this example, the employee must work from 8:00am to 4:00pm on Monday. Note that there will be an error if 0 employees are required at that time.

"Add Must Work" will add 1 employee to the table for all times designated as "Must Work" for this employee. "Subtract Must Work" subtracts 1 employee for those times.

Press OK to save the changes. Note that pressing Cancel will discard any changes made.

Create Schedule

When "Create Schedule" is clicked, the application tries to create shifts between the minimum and maximum specified, which can be edited:

The application favors creating longer shifts.

Sometimes the application is unable to create shifts that are at least the minimum length. In this example, the application must create a half-hour shift from 1:30-2:00 on Thursday.

There are 4 options when this problem arises.

1. Extend Shift

Make the shift longer. This will change the employee requirements to add an additional employee for the remaining time.

2. Remove Shift

Remove the shift. This will change the employee requirements to subtract one employee for the duration of the shift.

3. Leave As Is

Make no changes.

4. Cancel

Stop creating the schedule.

There is another error that can occur:

This happens when more employees "must work" than are supposed to work at a particular time. In this case, 3 employees "must work" at 7:00am on Sunday (as specified in "Set Availability"), but according to the table, only 2 employees are supposed to work at 7:00am on Sunday.

To fix this, either remove the "must work" designation from some employees by clicking "Set Availability" on them, or increase the number of employees in the table.

If there are no more problems, the schedule will appear in the box on the right.

Calendar View

To view the schedule as a calendar or to print the schedule, click Calendar View.

There are two fields under the button: Title and Start Date.

"Start Date" is the date of Sunday on the calendar (the start of the week). If "Start Date" is November 8, the calendar will start on November 1, like so:

"Title" is the title that will be printed on the top of the calendar if it is printed. To print the calendar, click "Print" on top of the calendar.

The application will attempt to only schedule each employee for 5 days, though availability shall take precedence.

No employee will have two shifts that start on the same day.

Every employee will have at least 8 hours in between shifts.

Saving and Loading

Use "Save To File" to save schedules and "Load From File" to retrieve them. Save schedules as .txt files.
